# Estruturas de Repetição com `for` em Python

Repositório com exercícios desenvolvidos para praticar estruturas de repetição utilizando o laço `for` em Python.

## Exercícios

### 1. Verificação de Números Ímpares
Programa que percorre números de 1 a 99 e identifica quais são ímpares.

### 2. Potência de Números
Programa que calcula o quadrado de números de 1 a 4.

### 3. Cálculo de Fatorial
Programa que calcula o fatorial de um número inteiro utilizando função e laço `for`.

### 4. Conversão de Fahrenheit para Celsius
Programa que converte temperaturas de Fahrenheit para Celsius no intervalo de 50°F até 150°F.

### 5. Fatorial de Vários Números
Programa que solicita vários números ao usuário e calcula o fatorial de cada um deles.

### 6. Sistema de Reajuste Salarial
Programa que realiza cadastro de funcionários, calcula reajustes salariais e exibe os novos salários conforme regras definidas.

### 7. Série Matemática
Programa que calcula o valor de uma sequência matemática utilizando divisões sucessivas e acumuladores.

## Assuntos praticados

- Laço `for`
- Estruturas de repetição
- Condicionais (`if`, `elif`, `else`)
- Funções
- Contadores
- Acumuladores
- Operadores matemáticos
- Potenciação
- Fatorial
- Porcentagem
- Conversão de temperatura
- Entrada e saída de dados
- Lógica de programação
